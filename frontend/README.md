# Frontend (React + Vite)

## Features
- Mobile-friendly UI.
- Tabs: **Watchlist** and **History**.
- Mark each movie as watched/unwatched and liked/disliked.
- AI Search Agent input box with OTT provider selection.

## Setup
```bash
cd frontend
npm install
```

Optional `.env`:
```env
VITE_API_BASE_URL=http://localhost:8000
```

## Run
```bash
npm run dev
```

## Build
```bash
npm run build
```
